Musician assistant
==================

Features
--------

### Songs

Song book

#### Attributes

- Title
- Artist
- Duration
- Time signature
- Tempo
- Notes
- Lyrics
- Audio
- MIDI
- Sheet music

### Set lists

List of songs

#### Attributes

- Title
- Date
- Events
  - Intro
  - Chain / Pause
  - Break(s)
  - Encore(s)
  - Outro
  - Notes
- Duration

### Tools

- Metronome
- Import/Export
- Google Drive
- MIDI control I/O
- OSC control I/O

### Modes

- Create/Edit
- Rehearsal
- Studio
- Live
